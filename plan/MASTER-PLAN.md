## Working Session Pattern

**Each Session (1-2 hours):**
1. Recap - Review last session
2. Build - Create next component
3. Show - Demo results
4. Discuss - Get feedback
5. Refine - Make adjustments
6. Approve - Get go-ahead
7. Preview - Outline next work
